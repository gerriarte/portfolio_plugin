Professional portfolio management plugin with Elementor integration for showcasing projects and work.

== Description ==

Portfolio Projects Manager is a comprehensive WordPress plugin designed to help you create and manage professional portfolios with seamless Elementor integration. Perfect for designers, developers, agencies, and creative professionals.

= Key Features =

* **Project Management**: Create, edit, and organize portfolio projects with detailed information
* **Category System**: Organize projects with custom categories and color coding
* **Elementor Integration**: Native Elementor widget for easy portfolio display
* **Responsive Design**: Mobile-first approach with modern, clean design
* **Modal Gallery**: Interactive project details with image galleries
* **Statistics Tracking**: View counts and like functionality
* **API Support**: REST API endpoints for custom integrations
* **Multilingual Ready**: Full internationalization support
* **Admin Interface**: Intuitive admin panel for easy management

= Getting Started =

1. Install and activate the plugin
2. Go to Portfolio > Categories to create project categories
3. Add your first project in Portfolio > Projects
4. Use the Portfolio Grid widget in Elementor to display your portfolio

== Installation ==

= Automatic Installation =

1. Go to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "Portfolio Projects Manager"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin files
2. Upload the `portfolio-plugin` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress

= Requirements =

* WordPress 5.0 or higher
* PHP 7.4 or higher
* MySQL 5.6 or higher
* Elementor (optional but recommended)

== Frequently Asked Questions ==

= Does this plugin work without Elementor? =

Yes, the plugin works independently, but Elementor integration provides the best experience for displaying portfolios.

= Can I customize the appearance? =

Yes, the plugin includes extensive customization options through the admin panel and CSS customization.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready and includes Spanish and English language files.

= Can I import existing portfolios? =

Currently, manual entry is required, but we're working on import functionality for future versions.

= Does the plugin track analytics? =

Yes, the plugin tracks basic statistics like view counts and likes for each project.
